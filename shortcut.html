م Shortcut Generator

Builds an iOS Shortcut that sends every bank SMS straight into iMaliya — so you don't have to assemble it by hand.
“Unsigned” is normal. A website can't sign a Shortcut or add it to your Apple account — only Apple can, and only when a shortcut is shared from the Shortcuts app on a device. To install this one you just enable Allow Untrusted Shortcuts once (step 1). It's safe — you can read every action before adding it.

★ Easiest: build it yourself (no import, no setting)

A shortcut you create is trusted automatically — skip all the signing/untrusted stuff. It's just one action:
1
Shortcuts app → ＋ (new) → Add Action → search “Open URLs”.
2
In the URL field type … then tap the field and insert the Shortcut Input variable right after the =.
3
Name it iMaliya Log → Done. Then add it to your Message automation (step 3 below).

1 · (Only if importing the file) allow untrusted

a
Open Shortcuts and run any shortcut once (this unlocks the setting).
b
On iOS 18 / 26 the toggle moved: Settings → Apps → Shortcuts → Allow Untrusted Shortcuts. (Older iOS: Settings → Shortcuts.) Turn it ON.

2 · Your iMaliya address

Open this page in Safari (not the home-screen app). “Send to Shortcuts” opens the share sheet — pick Shortcuts (or Save to Files, then tap the file). You'll see Add Untrusted Shortcut → tap it.

3 · Make it run on every bank SMS

a
Shortcuts → Automation → ＋ → Create Personal Automation → "Message" → Message Contains, and type your bank words (e.g. debited, credited, Fawri, خصم, شراء).
b
Set Run Immediately (turn off "Ask Before Running").
c
Add action Run Shortcut → choose iMaliya Log. Make sure the message is passed (Shortcut Input). Done.
Opening a link needs the phone unlocked, so this method logs the SMS the next time you wake/unlock the phone (it queues while locked). For truly-locked background capture you'd need the Scriptable method instead.

Manual fallback (no import)

If importing won't work on your iOS, build the Shortcut by hand with two actions:
1
URL Encode → input: Shortcut Input.
2
Open URLs → URL: … then insert the URL Encoded Text variable at the end.
